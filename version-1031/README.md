# 国产高清影视大全 静态电影网站

- 已解析影片数量：2000
- 已生成详情页数量：2000
- 分类页数量：10
- 首页、分类页、排行榜页、详情页均已插入百度统计代码。
- 播放器使用上传 JS 中提取的 HLS 播放源并通过 hls-vendor-dru42stk.js 初始化。
- 素材包中未包含 1.jpg 到 150.jpg 图片文件，页面已按规则引用顶级目录下的 JPG 路径；部署时将图片放在网站根目录即可显示。

## 本地预览

建议在解压目录运行静态服务器后访问：

```bash
python3 -m http.server 8080
```

然后打开 `http://localhost:8080/index.html`。
